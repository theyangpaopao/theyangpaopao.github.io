c++ 基础学习（1）——面向对象（OOP)

1、重要特性

抽象、封装和数据隐藏、多态、继承、代码的可重用性

2、类（class)

是一种将抽象转换为用户定义类型的C++工具，它将数据表示和操纵数据的方法组合成一个整洁的包。

属性——成员变量 行为——成员方法

类声明：以数据成员的方式描述数据部分，以成员函数（被称为方法）的方式描述共有接口。

类方法定义：描述如何实现类成员函数。

3、类的封装和和访问控制

private 和public 是访问控制修饰符

private 修饰的内容只能在本类中访问

Public 修饰的内容是公共的，谁都可以访问

protected 表示受保护的权限，继承中用的比较多，子类能够访问父类中的文件

举例：立方体类

4、构造函数和析构函数

构造函数和析构函数：

构造函数:

1、主要作用在于创建对象时为对象的成员属性赋值，构造函数由编译器自动调用,无须手动调用。

2、构造函数名和类名相同

3、支持重载

析构函数:

1、主要作用在于对象销毁前系统自动调用，执行一 些清理工作。

2、构造函数名和类名相同，前面加上~

2、不允许有多个参数，无参，不支持重载

构造函数和析构函数有编译器自动调用一次，无需手动调用

如果没有手动提供，系统会自动提供一个无参的构造、析构函数，并且编译器会强制执行。

构造函数的分类和调用：

构造函数的分类：

  • 分类方式1，有无参数： 有参构造、无参构造

    如有写了有参构造函数，系统就不会提供无参函数

  • 分类方式2，类型：普通构造、拷贝构造

在C++中，下面三种对象需要调用拷贝构造函数（有时也称“复制构造函数”）：

1、一个对象用于给另外一个对象进行初始化（常称为赋值初始化）；

2、一个对象作为函数参数，以值传递的方式传入函数体；

3、一个对象作为函数返回值，以值传递的方式从函数返回；

构造函数调用的两种方式：

  • 显示调用

  • 隐示调用

调用规则：

默认情况下，编译器至少会提供类的三个函数

1、默认构造函数（无参，函数体为空）

2、默认的析构函数（无参，函数体为空）

3、默认拷贝构造函数，对类中非静态成员属性进行简单的值拷贝

如果用户定义了普通构造（非拷贝构造），系统不会提供默认无参构造，但是会提供拷贝构造

如果用户定义了拷贝构造，系统不会提供任何构造函数

深拷贝和浅拷贝

首先区分一下什么是基本类型和和引用类型：

1）、基本类型：就是值类型，即在变量所对应的内存区域存储的是值

2）、引用类型：就是地址类型。

浅拷贝：单纯的值拷贝，引用类型赋值时，赋的值地址（就是引用类型变量在内存中保存的内容）

拷贝是用了统一个堆空间，在释放其中一个对象时，指针也会释放。造成另一个对象指针错误

深拷贝：重新创建堆空间

5、初始化列表

一种语法，用来对成员数据进行初始化

如果在一个类中有 const 类型，不能在调用构造函数进行该值初始化，所以在创建时进行初始化

对于引用类型的成员，也必须使用初始化列表语法进行初始化

6、类对象作为类成员

当其他类作为本类中的成员

构造的顺序是：先调用其他类的构造，再调用本类的构造

析构的顺序：和构造相反。

成员构造先后顺序是根据在类中的顺序决定的，跟构造方法的初始化顺序无关

7、静态成员

在类定义中，他的成员（成员变量，成员方法），这些成员都可以用关键字static，称之为静态成员。

特点：不管这个类创建了多少对象，静态成员只有一个拷贝。这个拷贝被所有属于这个类的对象所共享。

静态成员在编译阶段就定义了

静态成员变量：

1、必须在类中声明，类外定义

2、 静态成员不属于某个对象，而属于类，所有对象共享

3、静态成员变量可以通过类名或者对象来获取

静态成员函数：

1、静态成员函数不能访问非静态成员变量，但是可以访问非静态成员变量

2、普通的成员函数可以访问静态或者非静态的成员变量、成员函数

3、可以通过类名或者对象名访问

8、单例设计模式

单例：单个实例（对象），创建一个对象，实例化一个对象，创建一个类的实例

设计模式：程序员的经验总结

单例设计模式：一个类在一个系统中（程序中）只能有一个对象。

单例的好处：防止创建过多的对象，造成内存浪费。

10、友元

在类外能够访问一个类的私有成员

友元分为三种：

1、友元（全局）函数

2、友元类

3、友元成员函数

友元关系不能被继承。

友元的关系是单向的，类a是类b的friend，但类b不一定是类a的friend。

友元关系不具有传递性。

11、运算符重载

运算符重载是什么：

使对象操作更美观的技术。
运算符重载是一种形式的c++多态。
要重载运算符，需使用被称为运算符函数的特殊函数形式。

运算符函数的格式如下：

operatop( argument-list)

例如，[operator]+() 重载+运算符
   operator*()重载*运算符
op必须是有效的c++运算符，不能虚构一个新的符号

很多c++ 运算符已经被重载了，例如*运算符，被用为地址，得到存储在这个地址的值
